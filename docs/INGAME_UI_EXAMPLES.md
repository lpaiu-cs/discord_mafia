# 인게임 UI 예시

이 문서는 현재 엔진 구현 기준으로, 플레이어가 실제 게임에서 보게 되는 공개 채널/DM/UI 흐름을 예시로 정리한 것이다.

## 1. 로비 공개 채널

제목:

```text
마피아42 시즌4 로비
```

설명:

```text
규칙셋: 시즌4 밸런스
```

필드 예시:

```text
참가자 (5/8)
1. A (방장)
2. B
3. C
4. D
5. E

안내
4명 이상이 되면 방장이 시작할 수 있습니다. 개인 DM이 열려 있어야 게임을 진행할 수 있습니다.
```

버튼:

```text
[참가] [나가기] [시작]
```

## 2. 역할 카드 DM

### 스파이

제목:

```text
당신의 직업은 스파이입니다.
```

설명:

```text
밤마다 한 명의 직업을 확인합니다. 마피아를 맞히면 즉시 접선하고 같은 밤에 한 번 더 조사할 수 있습니다.
```

### 짐승인간

밸런스 규칙셋 설명:

```text
접선 전에는 밤마다 한 명을 표시합니다. 표시한 사람이 실제로 마피아에게 죽으면 접선합니다. 회피가 발동하면 즉시 접선합니다. 접선 후 다른 마피아팀 생존자가 없으면 혼자 대상을 처형할 수 있고, 이 공격은 의사/군인/연인 효과를 무시합니다.
```

### 영매

```text
죽은 사람의 채팅을 보고, 밤마다 죽은 사람 한 명의 직업을 알아내 성불시킵니다.
```

### 성직자

```text
한 번만 밤에 죽은 플레이어 한 명을 부활시킵니다.
```

## 3. 밤 DM 선택 UI

### 스파이 1차 조사

제목:

```text
스파이 조사
```

설명:

```text
조사할 플레이어 한 명을 선택하세요.
```

### 스파이 보너스 조사

마피아를 적중하면 같은 DM 메시지가 아래처럼 갱신된다.

제목:

```text
스파이 추가 조사
```

설명:

```text
첫 조사로 마피아를 찾아 접선에 성공했습니다. 마피아 채널: <#채널ID> 같은 밤에 한 번 더 조사할 수 있습니다.
```

필드:

```text
1차 선택: 마피아
2차 선택: 아직 선택하지 않음
```

셀렉트 placeholder:

```text
추가 조사 대상을 선택하세요
```

### 접선된 짐승인간 단독 킬

다른 마피아팀 생존자가 없을 때 제목이 `짐승인간 처형 대상`으로 바뀐다.

```text
마피아가 전멸한 뒤에는 혼자 대상을 처형할 수 있습니다.
```

### 영매 성불 DM

밤 사망 판정이 끝난 뒤, 성불되지 않은 망자가 있으면 영매에게 별도 DM이 간다.

제목:

```text
영매 성불
```

설명:

```text
죽은 플레이어 한 명을 골라 직업을 확인하고 성불시킵니다.
```

### 성직자 부활 DM

이번 밤 사망자가 있고 성직자를 아직 쓰지 않았다면, 밤 사망 판정 뒤 별도 DM이 간다.

제목:

```text
성직자 부활
```

설명:

```text
이번 밤에 죽은 플레이어 한 명을 골라 부활시킵니다.
```

### 유혹으로 밤 행동 불가

행동 직업이 마담에게 걸리면 DM으로 아래 문구가 간다.

```text
오늘 밤에는 유혹 상태라 능력을 사용할 수 없습니다.
```

## 4. 기자 기사 UI

밤 취재 성공 후 기자 DM:

```text
대상 님의 기사를 준비했습니다. 공개 가능 시점은 3번째 낮입니다.
```

공개 가능한 낮이 되면 기자 DM:

제목:

```text
기자 기사 공개
```

설명:

```text
대상 님의 기사를 준비했습니다. 낮 동안 직접 공개할 수 있습니다.
```

버튼:

```text
[기사 공개]
```

버튼을 누르면 공개 채널에 새 메시지가 올라간다.

제목:

```text
기자 기사
```

설명:

```text
기자 기사: 대상 님의 직업은 의사입니다.
```

## 5. 공개 채널 단계 메시지

밤 시작:

```text
1번째 밤
개인 DM으로 행동을 제출해 주세요. 공개 채널에서는 결과만 안내합니다.
```

낮 시작:

```text
1번째 낮
토론 시간입니다. 살아 있는 플레이어는 한 번씩 시간을 늘리거나 줄일 수 있습니다.
```

투표:

```text
투표 시간
드롭다운으로 한 명을 선택해 주세요.
```

찬반:

```text
찬반 투표
대상 님을 처형할지 결정합니다.
```

## 6. 비밀 채널 가시성

### 밤

- `mafia-team`: 살아 있는 마피아/접선자 쓰기 가능, 망자는 읽기만 가능
- `lovers`: 살아 있는 연인 쓰기 가능, 망자는 읽기만 가능
- `graveyard`: 죽은 플레이어와 영매만 보며, 성불하지 않은 망자와 영매만 쓰기 가능

### 낮

- `mafia-team`: 살아 있는 마피아/접선자만 읽기 가능, 쓰기 불가
- `lovers`: 살아 있는 연인만 읽기 가능, 쓰기 불가
- `graveyard`: 완전히 닫힘

## 7. 재판/판정 관련 체감 UI

- 정치인이 마담에게 유혹당하지 않았다면 찬반 직후 `정치인은 투표 처형되지 않습니다.`가 최근 알림에 뜬다.
- 유혹당한 정치인은 일반 플레이어처럼 처형된다.
- 연인 희생은 공개 채널에 별도 문구가 뜨지 않고, 살아남은 연인에게만 `OO 님이 당신을 노린 마피아였습니다.` DM이 간다.
