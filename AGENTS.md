# AGENTS.md

## 목적

이 저장소는 Discord 서버에서 구동되는 `마피아42 시즌4 일반·클래식 복제 봇`을 만든다.
현재 작업 범위는 다음으로 고정한다.

- `8인 이하`
- `일반/클래식`
- `예언자, 판사, 교주팀 제외`
- `듀얼 고유능력 제외`
- `룰방 로컬 규칙 제외`

## 소스 우선순위

규칙을 확인할 때는 반드시 아래 순서를 따른다.

1. `시즌4 업데이트 문서`
2. `당시와 충돌하지 않는 기본 규칙 문서`
3. `당시와 충돌하지 않는 직업 문서`
4. `해당 직업의 출시 공지`
5. 그래도 비면 `미확정`으로 남긴다

후대 문서에 있는 표현이나 효과를 시즌4 규칙처럼 섞어 쓰지 않는다.

## 시즌4 내부 시점 구분

시즌4 안에서도 규칙이 바뀌므로 최소 아래 두 구간을 구분한다.

- `S4-초기`: `2016-10-10 16:00` 이후, `2017-01-13 15:59` 이전
- `S4-밸런스`: `2017-01-13 16:00` 이후

참고:

- `성직자` 추가: `2016-10-10 16:00`
- `예언자`, `판사` 추가: `2017-03-24 16:00`
- 이 저장소는 `2017-03-24` 이후에도 `예언자`, `판사`를 범위 밖으로 유지한다.

## 현재 허용 직업

- 마피아팀: 마피아, 스파이, 짐승인간, 마담
- 시민팀: 경찰, 의사, 군인, 정치인, 영매, 연인, 건달, 기자, 탐정, 도굴꾼, 테러리스트, 성직자
- 이 저장소에서 `탐정`은 마피아42의 `사립탐정`을 뜻한다.

## 작업 원칙

- 게임 규칙의 단일 기준 문서는 `RULE.md`다.
- `RULE.md`에는 반드시 `확정`, `복원`, `미확정`을 구분해서 적는다.
- 시즌4 문서와 후대 직업 문서가 충돌하면 시즌4 문서를 우선한다.
- 후대 직업 문서는 시즌4 문서가 비워 둔 항목만 보강하는 용도로만 쓴다.
- 애매한 상호작용을 임의 해석해서 코드에 넣지 않는다.
- 서버 구현에 필요한 전역 우선순위표가 공개 자료만으로 확정되지 않으면 그 상태를 문서에 그대로 남긴다.

## Discord 구현 가이드

- Discord는 전달 수단일 뿐이며, 규칙 자체는 시즌4 기준을 우선한다.
- 비공개 정보는 DM 또는 그에 준하는 비공개 인터페이스로 전달한다.
- 공개 정보만 공개 채널에 노출한다.
- 플레이어용 문구와 명령어는 한국어 기준으로 설계한다.
